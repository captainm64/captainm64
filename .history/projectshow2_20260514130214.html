My Humidifier Control Project

This was a project I also made for Programming 1, this project doesn't receive input, and instead automatically gives outputs based on an array that is in its code.

Download the Project Here!

This program works by checking a selection of arrays inside of it, and then comparing the thresholds to the current temperatures and humidity to see if the sensor or sprinkler or both need to be turned on, before outputting the status of the humifier and temperature sensors for each of the 4 instantces.
